# AI Response Quality Evaluator

AI Response Quality Evaluator is a multi-agent system that evaluates AI-generated responses using Retrieval-Augmented Generation (RAG) and multiple evaluation agents.

## Features

- Question & AI Response Submission
- Optional Reference Answer
- Optional Source Document Upload
- RAG-based Knowledge Retrieval
- Multi-Agent Evaluation
- Relevance Score
- Accuracy Score
- Hallucination Detection
- Completeness Score
- Final Verdict
- Analytics Dashboard
